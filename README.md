#### react-antd-admin-server 是后端
#### react-antd-admin-client 是前端
### 使用仓库相关命令
#### 安装前端所有依赖
```bash
npm install
```
#### 启动前端项目
```bash
npm start
```
#### 安装后端所有依赖
```bash
npm install
```
#### 启动后端项目
```bash
npm start
```
